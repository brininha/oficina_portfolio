# 👩‍💻 Construção de portfólio web

Boas-vindas ao repositório da nossa oficina de criação de portfólios web!

---

## 🗂️ O que você vai encontrar aqui?

Este repositório está dividido em duas partes principais: **templates** (para você não precisar começar do zero) e **guias** (para te auxiliar a personalizar tudo do seu jeito).

### 🎨 1. Templates de portfólio
Navegue pelas pastas dentro de `templates` e escolha a estrutura que mais combina com você:

*   📁 **[template-1](./templates/portfolio01/)**: um layout estilo currículo, bem dividido em seções, ideal para um perfil mais corporativo.
*   📁 **[template-2](./templates/portfolio02/)**: inspirado em redes sociais, rápido de editar e ótimo para abrir no celular.
*   📁 **[template-3](./templates/portfolio03/)**: página única perfeita para focar na sua história e nos seus projetos.

### 📚 2. Guias e dicas
Na pasta `guias`, você encontra os nossos tutoriais passo a passo:

*   📖 **[Kits de estilo e cores](./guias/estilos.md)**: contém 4 kits de identidade visual prontos para você usar no seu CSS ou se inspirar.
*   🚀 **[Guia de deploy no GitHub Pages](./guias/deploy.md)**: passo a passo de como colocar o seu site no ar e gerar o seu link oficial.

---

## 🛠️ Como usar este repositório na oficina

Siga estes 4 passos para construir o seu portfólio hoje:

1.  **Faça um fork:** clique no botão `fork` para fazer uma cópia deste repositório para a sua própria conta do GitHub.
2.  **Abra o editor:** no seu novo repositório, aperte a tecla `.` (ponto) no seu teclado. Isso abrirá o VS Code diretamente no seu navegador.
3.  **Escolha e edite:** abra a pasta do template que você mais gostou e comece a alterar os textos no arquivo `index.html` e as cores no `style.css`.
4.  **Publique:** salve tudo fazendo um *commit* e siga o **[guia de deploy](./guias/deploy.md)** para colocar na internet.
